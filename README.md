# Food Management System Frontend

> [Sever Repository ➡️](https://github.com/VishnuKumarSS/Food-Management-System-Django-BE)

## Client side configurations

#### 1. Install npm packages:
```bash
npm i
```

#### 2. Run development server:
```bash
npm run dev
```
